# Think Ink

**Think Ink** is a multiplayer Pictionary-style game where one player draws and the other guesses. Draw fast, guess faster, and share the fun with friends! 🎨🖊️

## How to Play

1. Open `index.html` in a browser.
2. Choose **Start Game** or **Join Game**:
   - **Start Game**: Pick **Draw** or **Guess**. A game code is generated—share it with a friend.
   - **Join Game**: Enter the game code your friend shared. Your role is assigned automatically.
3. The game begins! Draw, guess, and have fun.
